# 📝 maro-plugin-notes

Custodia la memoria viva del equipo.

Este plugin extiende Maro con un círculo sagrado para las notas:  
un repositorio central donde el conocimiento se comparte, perdura y siembra automatizaciones futuras.

Deja atrás la niebla del olvido. Haz que cada aprendizaje sea colectivo y todo apunte hacia el flujo compartido.

---

**Comandos principales**
- Crear y editar notas accesibles a todo el equipo.
- Buscar y listar anotaciones por proyecto, fecha o tema.
- Abrir y actualizar notas desde la terminal.
- Preparar el terreno para automatizaciones impulsadas por conocimiento explícito.

Convoca este círculo para que el saber fluya y nunca más se pierda la chispa entre los pasajes de lo cotidiano.
